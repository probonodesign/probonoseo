## Overview

ProbonoSEO is a comprehensive SEO plugin designed and developed specifically for Japanese websites. Built from scratch with Japanese text structures and content patterns in mind.

**Why ProbonoSEO?**

- **Made in Japan** - Developed by Japanese developers who understand Japanese SEO
- **70 Free Features** - More free features than other SEO plugins
- **Japanese Optimized** - Properly handles Japanese titles and meta descriptions
- **One-time Purchase** - Pro version is not a subscription, buy once and use forever

## Installation

### From ProbonoSEO Website (Recommended)

1. Download the latest version from [seo.prbn.org](https://seo.prbn.org)
2. Go to WordPress Admin → Plugins → Add New → Upload Plugin
3. Upload the ZIP file and click "Install Now"
4. Activate the plugin
5. Open the ProbonoSEO menu in the admin dashboard

### From WordPress.org (Coming Soon)

WordPress.org review in progress. Once approved:

1. Go to WordPress Admin → Plugins → Add New
2. Search for "ProbonoSEO"
3. Click "Install Now" and activate

## Security

- Free version has no external connections
- No tracking or analytics
- No ads within the plugin
- Pro version connects to official server only for license verification
- AI features use site owner's own OpenAI API key
- Fully compliant with WordPress Plugin Check and WordPress coding standards

## Third Party Services

### License Verification (Pro only)
Connects to license server for Pro license activation.
- Service URL: https://seo.prbn.org/api/
- Data sent: License key, site URL
- [Privacy Policy](https://seo.prbn.org/privacy-policy/) | [Terms of Service](https://seo.prbn.org/terms/)

### OpenAI API (Pro only, optional)
AI-powered SEO features require user's own OpenAI API key.
- [OpenAI Privacy Policy](https://openai.com/privacy/) | [Terms of Service](https://openai.com/terms/)

The free version does not connect to any external services.
