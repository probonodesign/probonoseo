== Description ==

ProbonoSEO Pro は、日本語サイトのために設計・開発された総合SEOプラグインです。日本語の文章構造やコンテンツパターンを前提に、ゼロから構築しました。

= ProbonoSEO Pro を選ぶ理由 =

* **Made in Japan** - 日本のSEOを理解した日本人開発者が開発
* **全288機能** - AI搭載SEO補助を含む完全なSEOツールキット
* **日本語最適化** - 日本語のタイトル、メタディスクリプションを適切に処理
* **買い切り型** - サブスクではなく、一度の購入で永続利用

== Installation ==

1. seo.prbn.org からZIPファイルをダウンロード
2. プラグイン > 新規追加 > プラグインのアップロード からZIPファイルをアップロード
3. プラグイン一覧から「有効化」
4. 管理画面メニュー「ProbonoSEO」を開く
5. ライセンスタブでライセンスキーを入力して有効化

== Frequently Asked Questions ==

= 日本語サイト向けですか？ =
はい。ProbonoSEO は日本語サイト向けに設計・最適化されています。

= Made in Japan ですか？ =
はい。ProbonoSEO は日本国内で、日本人開発者によって開発されています。

= Pro版はサブスクですか？ =
いいえ。Proライセンスは買い切り型で、永続的なアップデートが含まれます。

= Basic版との違いは何ですか？ =
Basic版は70の無料機能を含みます。Pro版はBasic版の70機能すべてに加え、218のPro限定機能（合計288機能）を含みます。AI補助、競合分析、リッチスニペット、高度な診断機能などが利用できます。

= Basic版からPro版にアップグレードできますか？ =
はい。Basic版を無効化し、Pro版をインストール・有効化するだけです。設定は引き継がれます。
