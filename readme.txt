== Installation ==

1. Download the ZIP file from seo.prbn.org
2. Go to Plugins > Add New > Upload Plugin and upload the ZIP file
3. Activate the plugin from the Plugins list
4. Open the ProbonoSEO menu in the admin dashboard
5. Enter your license key in the License tab to activate

== Frequently Asked Questions ==

= Is this for Japanese websites? =
Yes. ProbonoSEO is designed and optimized for Japanese websites.

= Is it Made in Japan? =
Yes. ProbonoSEO is developed in Japan by Japanese developers.

= Is the Pro version a subscription? =
No. The Pro license is a one-time purchase with lifetime updates.

= What is the difference from Basic? =
Basic includes 70 free features. Pro includes all 70 Basic features plus 218 Pro-exclusive features (288 total), including AI assistance, competitor analysis, rich snippets, and advanced diagnostics.

= Can I upgrade from Basic to Pro? =
Yes. Simply deactivate Basic, install Pro, and activate it. Your settings will be preserved.
