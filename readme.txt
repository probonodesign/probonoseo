== Description ==

ProbonoSEO is a comprehensive SEO plugin designed and developed specifically for Japanese websites. Built from scratch with Japanese text structures and content patterns in mind.

= Why Choose ProbonoSEO =

* **Made in Japan** - Developed by Japanese developers who understand Japanese SEO
* **70 Free Features** - More free features than other SEO plugins
* **Japanese Optimized** - Properly handles Japanese titles and meta descriptions
* **One-time Purchase** - Pro version is not a subscription, buy once and use forever

= Free Features (70 features) =

**Title Optimization (7 features)**
Basic title tag optimization, separator settings, site name append, title and H1 consistency check, category name append, duplicate title check, symbol and emoji warnings

**Meta Description (7 features)**
Basic meta description optimization, automatic content extraction, keyword extraction, summary generation, forbidden phrase check, character count check, duplicate check

**Canonical (5 features)**
Basic canonical settings, automatic canonical generation, trailing slash normalization, www normalization, parameter removal

**OGP / Twitter (12 features)**
Basic OGP output, OGP title settings, OGP description settings, automatic OGP image, default OGP image, Facebook App ID, LINE compatible OGP, thumbnail generation, image size detection, image alt check, Japanese URL support, basic Twitter card output

**Schema / Breadcrumbs (16 features)**
Basic schema output (Article, WebSite, WebPage, Organization, Person, BreadcrumbList, SearchAction, ImageObject), basic breadcrumb output (home, category, tag, post, page, archive, search, 404)

**Internal Links / Speed (12 features)**
Previous/next post links, same category links, child page links, related post links, tag relation logic, external link nofollow, category link format, lazy load images, lazy load iframes, CSS minification, JS minification, WordPress script optimization

**Article SEO (6 features)**
Heading structure check, image alt check, image count check, word count check, category match check, tag duplicate check

**Site Diagnosis (3 features)**
Title duplicate diagnosis, meta description duplicate diagnosis, speed diagnosis

**Other (2 features)**
Meta element cleanup, Google Search Console verification code

= Third Party Services =

This plugin connects to external services in certain situations.

**License Verification (Pro only)**
When activating a Pro license, the plugin connects to our license server to verify the license key.

* Service URL: https://seo.prbn.org/api/
* Purpose: License activation and verification
* Data sent: License key, site URL
* Privacy Policy: https://seo.prbn.org/privacy-policy/
* Terms of Service: https://seo.prbn.org/terms/

**OpenAI API (Pro only, optional)**
AI-powered SEO features use the OpenAI API. This feature is optional and requires the user to enter their own API key.

* Service: OpenAI API
* Purpose: AI title suggestions, content analysis, FAQ generation
* Data sent: Post content (only when user executes AI features)
* Privacy Policy: https://openai.com/privacy/
* Terms of Service: https://openai.com/terms/

The free version does not connect to any external services.

== Installation ==

1. Go to Plugins > Add New > Upload Plugin and upload the ZIP file
2. Activate the plugin from the Plugins list
3. Open the ProbonoSEO menu in the admin dashboard
4. Basic SEO features work automatically
5. For Pro features, enter your license key in the License tab

== Frequently Asked Questions ==

= Is this for Japanese websites? =
Yes. ProbonoSEO is designed and optimized for Japanese websites. It handles Japanese text structures and follows SEO best practices for Japanese search engines.

= Is it Made in Japan? =
Yes. ProbonoSEO is developed in Japan by Japanese developers.

= Can I use it with other SEO plugins? =
Possible but not recommended. ProbonoSEO automatically detects other SEO plugins (Yoast, RankMath, etc.) and suppresses duplicate output to avoid conflicts.

= Does it work with any WordPress theme? =
Yes. It works with any standard WordPress theme. Highly customized themes may require minor adjustments.

= What schema types are included in the free version? =
The free version includes: WebSite, WebPage, Organization, Person, Article, BreadcrumbList, SearchAction, ImageObject

= How do I access Pro features? =
Purchase a Pro license at [https://seo.prbn.org](https://seo.prbn.org) and enter your license key in ProbonoSEO > License tab.

= Is the Pro version a subscription? =
No. The Pro license is a one-time purchase with lifetime updates.

= Does this plugin collect user data? =
The free version does not collect data or connect to external services. The Pro version connects to our license server for verification and optionally to OpenAI API for AI features. See "Third Party Services" section for details.
